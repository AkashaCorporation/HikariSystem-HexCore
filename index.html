// capabilities

Built for Security Researchers

Everything you need for binary analysis in one unified environment.

🔍

Disassembly

Native multi-arch disassembler (x86, x64, ARM, ARM64, MIPS, RISC-V) with function detection, string cross-references, and IDA-style graph view.

🎮

CPU Emulation

Full PE and ELF emulation via Unicorn Engine. 65+ API hooks (Windows + Linux), stdin emulation, TLS support, and syscall handling.

🧬

IR Lifting NEW

Machine code to LLVM IR translation via Remill engine. Analyze binary semantics at the intermediate representation level.

🛡

YARA Scanner v2.1

Real YARA rule parsing with DefenderYara integration. 76,000+ Microsoft Defender signatures, threat scoring, headless pipeline support.

🌍

IOC Extractor NEW

Binary-aware IOC detection: IPs, URLs, domains, named pipes, crypto wallets. SQLite backend with noise reduction and threat assessment.

📝

Minidump Parser NEW

Windows crash dump forensics with thread/module/memory parsing. Thread injection and RWX detection with threat heuristics.

🔧

Assembly Patching

Inline binary patching with LLVM MC backend. NOP sleds, instruction replacement, multi-architecture support.

📊

Entropy Analysis

Streaming entropy analysis with adaptive block sizing. Identifies packers, cryptors, and obfuscated sections with modular report pipeline.

🔏

Strings & XOR v1.2

Memory-efficient string extraction with XOR deobfuscation and stack string detection. Auto-categorization and cross-references.

⚙

Automation Pipeline

Headless batch analysis via JSON job files. Auto-trigger, per-step timeout, extension preflight, and structured output reports.

💾

Hex Viewer

Professional binary viewer with virtual scrolling, data inspector, bookmarks, structure templates, and hex pattern search.

🧮

Constant Decoder

Hover any immediate value in disassembly to see Hex, Unsigned, Signed32/64, Binary, ASCII, and Float32 representations.

// native engines

N-API Powered

Native C/C++ engines compiled as Node.js addons. No external installs required.

hexcore-capstone

Capstone v5 disassembly engine. Async mode, detail output, all architectures.

hexcore-unicorn

Unicorn CPU emulator. Hooks, context save/restore, multi-architecture.

hexcore-llvm-mc

LLVM 18.1.8 MC assembler. Multi-arch assembly and binary patching.

hexcore-remill

Remill IR lifting engine. Machine code to LLVM IR translation. 168 static libs.

hexcore-better-sqlite3

SQLite N-API wrapper for IOC persistence, caching, and structured data storage.

// getting started

Installation

1

Clone the repository

git clone https://github.com/LXrdKnowkill/HikariSystem-HexCore.git
2

Install dependencies

cd HikariSystem-HexCore && npm install
3

Launch HexCore

$env:VSCODE_SKIP_NODE_VERSION_CHECK="1"; .\scripts\code.bat
4

Open a binary and start analyzing

Right-click any PE/ELF file to disassemble, emulate, scan with YARA, or extract IOCs.
